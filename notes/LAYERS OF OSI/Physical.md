# PHYSICAL LAYER (Layer 1)

This layer deals with the raw physical bits.

## Key Concepts
**Bits** → Signals

## Encoding Methods
### Manchester Encoding
At the middle of the clock cycle:
- If the curve is going **up** → means **1**
- If the curve is going **down** → means **0**
